# Gauras Farm Real Photos Upload Directory

Place your real farm photos in this directory (`/home/siddharth/annapurna-web/images/`) with the following filenames to integrate them into the website automatically:

1. **`hero.jpg`** (or `.png` / `.webp`): The main cover photo at the top of the homepage (replaces the Unsplash dairy farm landscape photo).
2. **`sahiwal.jpg`**: Replaces the Sahiwal crossbreed cow profile photo.
3. **`murrah.jpg`**: Replaces the Murrah buffalo profile photo.
4. **`machinery.jpg`**: Replaces the milking machinery photo.
5. **`vet.jpg`**: Replaces the veterinary care photo.

---

### Note on File Formats
If you upload images in another format (like `.png` or `.webp`) instead of `.jpg`, please let me know so I can update the file extensions in `index.html` for you, or save them directly with `.jpg` file extension!
